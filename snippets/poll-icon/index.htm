You now need to add the imageset to your forum. To do this, go to your ACP -> Styles Tab ->  Imagesets ->  Edit Prosilver ->  Select "Poll" 
from the drop down (under General icons). On the image section below that, select "icon_topic_poll.gif" from that dropdown, select yes to include dimensions but leave the boxes blank and then click submit.
