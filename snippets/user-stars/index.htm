User Stars

Admins can give any user a 'Star' that will show next to their username

The Star is an image that will show next to their username in viewtopic and profiles. This Star can be used as a reward system - good users get a Star. Or for marking donators to your site. You can set, per user, text that will show up when you hover over the users star image.

Some of the features:

  • Turn on/off per-user
  • User Stars to show in these locations by default -
    • Viewtopic
    • Viewprofile
    • Memberlist/Profile
    • Viewonline page
    • Who's Online (Index/Forum)
    • Friends/Foes UCP Block
    • Groups/Teams page

ACP features:

  • New field in ACP section where admins can define the location of the Star image
  • New field in ACP section where admins can define text to be show when someone hovers over the Star image. (ie. 'UserX is Power Member', 'UserX has donated to foo.com', 'UserX is a donator')
